Reinforcement Learning Chess Bot (Knightmare Protocol)

Python PyTorch

This project investigates reinforcement learning for chess with an emphasis on learning from expert demonstrations.
I began by implementing NFQ and DQN on a reduced 5×5 board to study learning dynamics and stability, then scaled to full 8×8 chess using Double DQN and dueling architectures.
A supervised baseline was trained on approximately 8 million master-level positions and deployed on Lichess, achieving ~1600 Blitz Elo.
Current work focuses on transferring supervised representations into reinforcement learning to improve sample efficiency and playing strength.

Adversarial Robustness in Deep Learning

Python PyTorch

Worked as a student researcher under faculty supervision on adversarial robustness in deep learning models.
The project involved using variational autoencoders to generate adversarial examples that fool convolutional neural networks, and evaluating their impact on model performance.
I then explored adversarial training strategies to improve robustness against these learned perturbations.

BERT-CRF Model for Attribute-Value Extraction

Python NLP

Built an end-to-end system for attribute–value extraction from unstructured e-commerce product titles, framing the task as sequence labeling with IOB tags.
Implemented a BERT–CRF architecture that combines contextualized representations from a pre-trained transformer with structured decoding via a conditional random field, enabling valid and consistent tag sequences.
The system includes a full training pipeline and an interactive web application for real-time inference and demonstration.
